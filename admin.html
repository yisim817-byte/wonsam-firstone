개인 사전의향서, 기업 의향서, 기업자료 요청은 yisim817@gmail.com으로 수신됩니다.

관리 방법

  1. Gmail에서 [원삼 센트레빌] 제목으로 검색
  2. 개인 사전의향서 / 기업 의향서 / 기업자료 요청 제목별 분류
  3. 필요 시 Gmail 라벨을 만들어 관리

DB 조회 보류 안내

Supabase DB 조회 API는 유지되어 있으며, 향후 Vercel 환경변수 SUPABASE_URL, SUPABASE_SERVICE_ROLE_KEY, ADMIN_TOKEN 설정 후 재활성화할 수 있습니다. 현재 관리자 페이지에는 실제 접수자 개인정보 목록을 노출하지 않습니다.

Gmail 열기 접수 테스트: 개인 사전의향서 접수 테스트: 기업 의향서 접수 테스트: 기업자료 요청

광고홍보자료 관리

기업제안서·고객제안서를 업로드하면 홍보자료 페이지에 즉시 반영됩니다. 카테고리별로 새 자료를 게시하면 이전 게시본은 자동으로 비공개 처리됩니다.

PDF 업로드를 권장합니다. PPT/PPTX 업로드 시 서버에서 PDF 자동 변환을 시도하며, 변환 환경이 설정되지 않았거나 실패하면 프레젠테이션 원본을 PDF로 변환한 후 다시 업로드해 달라는 안내가 표시됩니다.

관리자 확인 후 광고홍보자료 관리 화면이 표시됩니다.
